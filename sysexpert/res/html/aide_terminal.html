Aide Terminal

L'interface est constituée d'un afficheur qui donne le résultat des commandes et de lignes de saisi qui permettent de rentrer des commandes. Pour en valider une, appuyez sur entrer.

Liste des commandes

Commandes Description
Aide Affiche l'aide
Afficher [faits|règles] Affiche les faits et/ou les rêgles
Règle [plus|complexe|première] Modifie le type de sélection des règles
Trace [non|min|oui] Modifie le type d'affichage lors du chaînage
Lire nom_du_fichier.txt Charge un fichier
Log Affiche le log
Réinitialise Réinitialiser la Base de connaissances

Autres

Il est possible d'entrer des règles et des faits avec la même syntaxe que les fichiers (Voir l'onglet "Fichier").
