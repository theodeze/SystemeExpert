Aide Terminal

L'interface est constitué d'un afficheur qui affiche le résultat des commandes et de ligne de saisi qui permet de rentrer des commandes, pour valider une commande appuyer sur entrer.

Liste des commandes

Commande Description
aide Affiche l'aide
afficher [faits|regles] Affiche les faits et/ou les rêgles
regle [plus|complexe|premiere] Modifie le type de selection des rêgles
trace [non|min|oui] Modifie le type d'affichage lors du chainage
lire nom_du_fichier.txt Charge un fichier
reinitialise Reinitialiser la Base de connaissance

Autre

Il est possible d'entre des règles et des faits avec la même syntaxe que les fichiers. (Voir l'onglet "Fichier")
